Projet React
============================================================
Réalisation d'une application Rich Media (multimédia, interactive & synchronisée) à l'aide de React - IMR3 - ENSSAT 2021

Run app
---
1. `npm install`
2. `npm start`
3. Connect to `http://localhost:3000/`

Run test
---
* `npm test`

Live demo
---------
https://react-ndelvoye.herokuapp.com/
